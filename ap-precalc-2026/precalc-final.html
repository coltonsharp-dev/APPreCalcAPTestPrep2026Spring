1A · Topic 1.2 Avg Rate of Change
A function f satisfies f(2) = 5 and f(7) = 20. What is the average rate of change of f on the interval [2, 7]?
A−3
B13
C5
D259

F15
Explanation

1A · Topic 1.3 Concavity
A continuous function f is increasing on [0, 10]. On the interval (0, 5), f is concave down; on the interval (5, 10), f is concave up. What feature does the graph have at x = 5?
ALocal maximum
BLocal minimum
CPoint of inflection
DVertical asymptote
EHorizontal asymptote
FRemovable discontinuity
Explanation
Concavity changes (∩ → ∪) but the function keeps increasing — so there is no direction change. Concavity flip with no direction flip = point of inflection, not an extremum.

1B · Topic 1.4 🔥 Biggest Miss
The graph of a polynomial f is increasing on the entire interval (1, 7). Between x = 3 and x = 5 the curve has an obvious S-bend where the concavity switches from concave up to concave down at x = 4. The point at x = 4 is best classified as:
ALocal maximum (the curve reaches a peak there)
BLocal minimum (the curve reaches a valley there)
CPoint of inflection (concavity changes; direction does not)
DGlobal maximum on the interval
EBoth a local max and a point of inflection
FA vertical asymptote
Explanation
This is the exact misconception your class missed on Practice Exam Q2. f is increasing throughout — it never changes direction — so x = 4 cannot be an extremum. Concavity alone changing means point of inflection.

1B · Topic 1.6 End Behavior
For the polynomial f(x) = −2x⁴ + 5x³ − x + 9, which statement correctly describes the end behavior?
AAs x → ±∞, f(x) → +∞
BAs x → −∞, f(x) → −∞  and  as x → +∞, f(x) → +∞
CAs x → −∞, f(x) → +∞  and  as x → +∞, f(x) → −∞
DAs x → ±∞, f(x) → −∞
EAs x → ±∞, f(x) → 0
FEnd behavior cannot be determined without more information
Explanation
End behavior depends only on the leading term: −2x⁴. Even degree (4) means both ends do the same thing; negative leading coefficient (−2) flips them downward. So both ends → −∞.

For the polynomial f(x) = 3(x + 2)²(x − 1)(x − 4)³, identify the behavior of the graph at each x-intercept.
Ax = −2 S-crosses, x = 1 bounces, x = 4 crosses cleanly
Bx = −2 crosses cleanly, x = 1 bounces, x = 4 S-crosses

Explanation

1C · Topic 1.7 Holes vs Asymptotes
For the rational function r(x) = (x − 3)(x + 1) / [(x − 3)(x + 5)], identify all holes and vertical asymptotes.
AVertical asymptotes at x = 3 and x = −5; hole at x = −1
BVertical asymptote at x = −5; hole at x = 3
CHoles at x = 3 and at x = −5
DVertical asymptotes at x = 3 and x = −5 (no holes)
EVertical asymptote at x = 3; hole at x = −5
FNo holes; vertical asymptote at x = −1 only
Explanation
The factor (x − 3) cancels from numerator and denominator → hole at x = 3. The factor (x + 5) only appears in the denominator → vertical asymptote at x = −5. Cancel first; classify the leftover roots.

Which statement best describes the long-run behavior of f(x) = (4x³ + 1) / (x² + 5)?
AHorizontal asymptote at y = 0
BHorizontal asymptote at y = 4
CHorizontal asymptote at y = 1/5

2B · Topic 2.12 Log Properties
Which expression is equivalent to log(x²√y / z³), where the √ denotes square root and all variables are positive?
A2 log x + (1/2) log y − 3 log z
B2 log x · (1/2) log y / 3 log z
Clog(2x) + log(√y) − log(3z)
D2 log x + log(½) · log y − 3 log z
E2 log x + (1/2) log y + 3 log z
Flog(2x + √y − 3z)
Explanation
Use product rule (multiplication → addition), quotient rule (division → subtraction), and power rule (exponent moves to a coefficient). Note √y = y^(1/2). The final expression is 2 log x + (1/2) log y − 3 log z.

2B · Topic 2.13 Log Equation
Solve for x: log₄(x) + log₄(x − 6) = 2.
Ax = 2
Bx = 4
Cx = 6
Dx = 8
Ex = 16
Fx = 22
Explanation
Combine: log₄(x(x − 6)) = 2 → x(x − 6) = 4² = 16 → x² − 6x − 16 = 0 → (x − 8)(x + 2) = 0. Reject x = −2 (log undefined). Answer: x = 8.

2B · Topic 2.15 🔥 Semi-log · 4/10 wrong
When the points (x, y) of a data set are graphed as (x, log₁₀ y) on a semi-log plot, the result is a line with slope 0.5 and y-intercept 1. Which formula best models the original function y(x)?
Ay = 0.5 x + 1  (simple linear)
By = log(0.5 x + 1)
Cy = 0.5 · 10^x
Dy = x^(0.5) + 10
Ey = 10 · 10^(0.5 x)
Fy = e^(0.5 x + 1)
Explanation
The line in semi-log space is log y = 0.5 x + 1. Exponentiate both sides (base 10): y = 10^(0.5x + 1) = 10¹ · 10^(0.5x) = 10 · 10^(0.5x). A linear semi-log plot ⇒ exponential original.

2C · Topic 2.8 🔥 Highest Miss in Unit 2
The graph of f passes through the points (1, 4), (3, 7), and (5, 12). Through which set of points must the graph of f⁻¹ (the inverse of f) pass?
A(1, ¼), (3, 1/7), (5, 1/12)
B(−1, −4), (−3, −7), (−5, −12)
C(1, 4), (3, 7), (5, 12)
D(4, −1), (7, −3), (12, −5)
E(¼, 1), (1/7, 3), (1/12, 5)
F(4, 1), (7, 3), (12, 5)
Explanation
The inverse swaps x and y in every ordered pair. (1, 4) → (4, 1), and so on. The inverse is a reflection across y = x, NOT a reciprocal (option A) and NOT a negation (option B).

3B · Topic 3.7 Sinusoidal Modeling
A Ferris wheel has a diameter of 60 m and its center is 35 m above the ground. It completes one full revolution every 4 minutes. A rider boards at the lowest point at time t = 0. Which equation models the rider's height h(t) above the ground (in meters)?
Ah(t) = 30·sin(πt/2) + 35
Bh(t) = 60·cos(πt/4) + 35
Ch(t) = 30·cos(πt/2) + 35
Dh(t) = −30·cos(πt/2) + 35
Eh(t) = 35·sin(πt/2) + 30
Fh(t) = −60·sin(πt/4) + 35
Explanation
Amplitude = radius = 30. Midline D = center height = 35. Period = 4 → B = 2π/4 = π/2. Boarding at the lowest point at t = 0 → use −cos (so it starts at the minimum). Together: −30·cos(πt/2) + 35.

4A · Topic 4.1 Find Parameter
A parametric curve is defined by x(t) = t² − 1 and y(t) = 2t + 3. For which value of t is the point on the curve equal to (3, 7)?
At = 2
Bt = 1
Ct = −2
Dt = 3
EBoth t = 2 and t = −2
Ft = 4
Explanation
From x: t² − 1 = 3 → t² = 4 → t = ±2. You must check y for each candidate. y(2) = 7 ✓ but y(−2) = −1 ✗. Only t = 2 produces both coordinates correctly. Option E is the trap: solving x alone.

Ax = 5 + 2 cos(t),   y = 5 − 3 sin(t),   0 ≤ t < 2π
Bx = 2 cos(t),   y = −3 sin(t),   0 ≤ t < 2π

How many points of intersection are there between the implicit relation x² + y² = 25 and the line y = x + 1?
A0
B1
C3
D4
EInfinitely many
F2
Explanation
Substitute y = x + 1 into the circle: x² + (x + 1)² = 25 → 2x² + 2x − 24 = 0 → x² + x − 12 = 0. Discriminant = 1 + 48 = 49 > 0 → two real solutions. The line is a chord that passes through the interior of the circle.

4C · Topic 4.6 Ellipse Standard Form
An ellipse is centered at (1, −2). Its major axis is horizontal with total length 10, and its minor axis is vertical with total length 6. Which equation represents this ellipse in standard form?
A(x + 1)²/25 + (y − 2)²/9 = 1
B(x − 1)²/25 + (y + 2)²/9 = 1
C(x − 1)²/100 + (y + 2)²/36 = 1
D(x − 1)²/9 + (y + 2)²/25 = 1
E(x − 1)²/10 + (y + 2)²/6 = 1
F(x − 1)²/5 + (y + 2)²/3 = 1
Explanation
Major axis length = 2a = 10 → a = 5 → a² = 25. Minor axis length = 2b = 6 → b = 3 → b² = 9. Center (1, −2) → (x − 1) and (y + 2). Major axis horizontal → larger denominator (25) goes under the x term. Common error: putting the lengths (10, 6) directly in denominators (option E).

4C · Topic 4.7 Parametric Conic
The parametric curve x = 4 cos(t),   y = 3 sin(t),   0 ≤ t < 2π, traces:
AA circle of radius 4
BAn ellipse with semi-major axis 4 (horizontal) and semi-minor axis 3 (vertical)
CA circle of radius 3

Explanation
