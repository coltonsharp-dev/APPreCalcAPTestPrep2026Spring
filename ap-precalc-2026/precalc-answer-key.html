1A · Topic 1.2 Avg Rate of Change
Answer E
A function f satisfies f(2) = 5 and f(7) = 20. What is the average rate of change of f on the interval [2, 7]?
A−3
B13
C5
D259
E3
F15
Worked Solution
AROC on [a, b] = (f(b) − f(a)) / (b − a). Plug in: (f(7) − f(2)) / (7 − 2) = (20 − 5) / 5 = 15 / 5 = 3. Common error: stopping at 15 (forgetting to divide by Δx).

1A · Topic 1.3 Concavity
Answer C
A continuous function f is increasing on [0, 10]. On the interval (0, 5), f is concave down; on the interval (5, 10), f is concave up. What feature does the graph have at x = 5?
ALocal maximum
BLocal minimum
CPoint of inflection
DVertical asymptote
EHorizontal asymptote
FRemovable discontinuity
Worked Solution
f is increasing on [0, 10], so the function never changes direction — there's no max or min. Concavity flips from concave-down (∩) to concave-up (∪) at x = 5. Concavity flip with no direction flip = point of inflection.

1B · Topic 1.4 🔥 Biggest Miss
Answer C
The graph of a polynomial f is increasing on the entire interval (1, 7). Between x = 3 and x = 5 the curve has an obvious S-bend where the concavity switches from concave up to concave down at x = 4. The point at x = 4 is best classified as:
ALocal maximum (the curve reaches a peak there)
BLocal minimum (the curve reaches a valley there)
CPoint of inflection (concavity changes; direction does not)
DGlobal maximum on the interval
EBoth a local max and a point of inflection
FA vertical asymptote
Worked Solution
This is the exact misconception your class missed on Practice Exam Q2. f is increasing throughout (1, 7), so the curve never reaches a peak or valley — extrema are impossible. Concavity changing alone (concave-up → concave-down) means a point of inflection, not a max.

1B · Topic 1.6 End Behavior
Answer D
For the polynomial f(x) = −2x⁴ + 5x³ − x + 9, which statement correctly describes the end behavior?
AAs x → ±∞, f(x) → +∞
BAs x → −∞, f(x) → −∞  and  as x → +∞, f(x) → +∞
CAs x → −∞, f(x) → +∞  and  as x → +∞, f(x) → −∞
DAs x → ±∞, f(x) → −∞
EAs x → ±∞, f(x) → 0
FEnd behavior cannot be determined without more information
Worked Solution
End behavior is determined by the leading term. Even degree (4) with negative leading coefficient (−2) ⇒ both ends point down. x → −∞ ⇒ f(x) → −∞   AND   x → +∞ ⇒ f(x) → −∞.

1B · Topic 1.5 Multiplicity
Answer F
For the polynomial f(x) = 3(x + 2)²(x − 1)(x − 4)³, identify the behavior of the graph at each x-intercept.
Ax = −2 S-crosses, x = 1 bounces, x = 4 crosses cleanly
Bx = −2 crosses cleanly, x = 1 bounces, x = 4 S-crosses
CAll three intercepts cross cleanly
Dx = −2 bounces, x = 1 S-crosses, x = 4 crosses cleanly
Ex = −2 bounces, x = 1 crosses cleanly, x = 4 bounces
Fx = −2 bounces, x = 1 crosses cleanly, x = 4 S-crosses
Worked Solution
Multiplicity dictates behavior at each zero of f(x) = 3(x + 2)²(x − 1)(x − 4)³: x = −2 (mult 2) bounces, x = 1 (mult 1) crosses cleanly, x = 4 (mult 3) S-curve crosses. Even multiplicity = bounce; odd ≥ 3 = S-shape cross.

1C · Topic 1.7 Holes vs Asymptotes
Answer B
For the rational function r(x) = (x − 3)(x + 1) / [(x − 3)(x + 5)], identify all holes and vertical asymptotes.
AVertical asymptotes at x = 3 and x = −5; hole at x = −1
BVertical asymptote at x = −5; hole at x = 3
CHoles at x = 3 and at x = −5
DVertical asymptotes at x = 3 and x = −5 (no holes)
EVertical asymptote at x = 3; hole at x = −5
FNo holes; vertical asymptote at x = −1 only
Worked Solution
Factor: numerator (x − 3)(x + 2), denominator (x + 5)(x − 3). x = 3 zeroes both top and bottom ⇒ removable hole at x = 3. x = −5 zeroes only the denominator ⇒ vertical asymptote at x = −5.

1C · Topic 1.8 Asymptotes
Answer E
Which statement best describes the long-run behavior of f(x) = (4x³ + 1) / (x² + 5)?
AHorizontal asymptote at y = 0
BHorizontal asymptote at y = 4
CHorizontal asymptote at y = 1/5
DHorizontal asymptote at y = 4x
ENo horizontal asymptote — the function has a slant (oblique) asymptote
FHorizontal asymptote at y = ∞
Worked Solution
Compare degrees of numerator and denominator. Numerator has degree 2, denominator has degree 1. Since deg(num) > deg(den), there is no horizontal asymptote. Specifically (deg num) − (deg den) = 1, so there's a slant (oblique) asymptote instead.

2B · Topic 2.12 Log Properties
Answer A
Which expression is equivalent to log(x²√y / z³), where the √ denotes square root and all variables are positive?
A2 log x + (1/2) log y − 3 log z
B2 log x · (1/2) log y / 3 log z
Clog(2x) + log(√y) − log(3z)
D2 log x + log(½) · log y − 3 log z
E2 log x + (1/2) log y + 3 log z
Flog(2x + √y − 3z)
Worked Solution
Apply the three log rules: product (× → +), quotient (÷ → −), power (exponent → coefficient). Note √y = y^(1/2). log(x²√y / z³) = log(x²) + log(y^(1/2)) − log(z³) = 2 log x + (1/2) log y − 3 log z.

When the points (x, y) of a data set are graphed as (x, log₁₀ y) on a semi-log plot, the result is a line with slope 0.5 and y-intercept 1. Which formula best models the original function y(x)?
Ay = 0.5 x + 1  (simple linear)

The graph of f passes through the points (1, 4), (3, 7), and (5, 12). Through which set of points must the graph of f⁻¹ (the inverse of f) pass?
A(1, ¼), (3, 1/7), (5, 1/12)

Answer D
A Ferris wheel has a diameter of 60 m and its center is 35 m above the ground. It completes one full revolution every 4 minutes. A rider boards at the lowest point at time t = 0. Which equation models the rider's height h(t) above the ground (in meters)?
Ah(t) = 30·sin(πt/2) + 35

Answer A
A parametric curve is defined by x(t) = t² − 1 and y(t) = 2t + 3. For which value of t is the point on the curve equal to (3, 7)?
At = 2
Bt = 1
Ct = −2
Dt = 3
EBoth t = 2 and t = −2
Ft = 4
Worked Solution

The parametric curve x = 4 cos(t),   y = 3 sin(t),   0 ≤ t < 2π, traces:

3B · Topic 3.6 / 3.10 Sinusoidal Modeling · 4 sub-parts
Tidal harbor: water level is sinusoidal. Maximum 22 ft at t = 6 hr, minimum 6 ft at t = 18 hr (t in hours after midnight). Find the model and evaluate.
(a) Find the amplitude A and the midline D.
Answer A = 8,   D = 14
Work
Amplitude is half the vertical range: A = (max − min) / 2 = (22 − 6) / 2 = 8. Midline is the average of max and min: D = (max + min) / 2 = (22 + 6) / 2 = 14.
(b) Find the period T (in hours) and the value of B.
Answer Period = 24 hours,   B = π/12
Work
From a maximum at t = 6 to a minimum at t = 18 is half of a full period: 12 hours = T/2 ⇒ T = 24 hours. Then B = 2π / T = 2π / 24 = π/12.
(c) Write the full sinusoidal equation h(t) using cosine.
Answer h(t) = 8 cos((π/12)(t − 6)) + 14
Work
Cosine is at its maximum when its argument is 0. We want the max at t = 6, so the phase shift is C = 6: cos((π/12)(t − 6)) peaks at t = 6. Multiply by amplitude 8 and add midline 14: h(t) = 8 cos((π/12)(t − 6)) + 14.
(d) Evaluate h(12).
Answer h(12) = 14 ft (the midline)
Work
h(12) = 8 cos((π/12)(12 − 6)) + 14 = 8 cos(π/2) + 14. cos(π/2) = 0, so h(12) = 8(0) + 14 = 14 ft. Geometrically, t = 12 is a quarter-period after the maximum at t = 6, so the function is crossing the midline going downward.

4C · Topic 4.7 / 4.10 Hyperbola → Parametric · 3 sub-parts
Hyperbola: (x − 1)² / 9 − (y + 2)² / 16 = 1. Identify key features and a parametric form.
(a) Identify the center and the two vertices.
Answer Center: (1, −2).   Vertices: (4, −2) and (−2, −2)
Work
Standard form: (x − h)²/a² − (y − k)²/b² = 1,   horizontal-opening. Read off h = 1, k = −2, a² = 9 ⇒ a = 3, b² = 16 ⇒ b = 4. Center = (h, k) = (1, −2). Vertices on the horizontal axis through the center, distance a = 3 left and right: (1 + 3, −2) = (4, −2) and (1 − 3, −2) = (−2, −2).
(b) Write the equations of the two asymptotes (slope-intercept form).
Answer y = (4/3)(x − 1) − 2   and   y = −(4/3)(x − 1) − 2
Work
Asymptote formula for horizontal hyperbola: y − k = ±(b/a)(x − h). With h = 1, k = −2, b/a = 4/3: y = ±(4/3)(x − 1) − 2. Either ordering accepted.
(c) Give a parametric form using sec(t) and tan(t), then evaluate at t = 0.
Answer x(t) = 1 + 3 sec(t),   y(t) = −2 + 4 tan(t).   At t = 0: (4, −2)
Work
Standard parametric form for a horizontal hyperbola: x = h + a sec(t),   y = k + b tan(t) (uses the identity sec²t − tan²t = 1, which mirrors x²/a² − y²/b² = 1). Plug in: x(t) = 1 + 3 sec(t),   y(t) = −2 + 4 tan(t). At t = 0: sec(0) = 1, tan(0) = 0 ⇒ x = 1 + 3 = 4, y = −2 + 0 = −2 ⇒ (4, −2) (which is the right vertex — the parametric form starts on a vertex).
