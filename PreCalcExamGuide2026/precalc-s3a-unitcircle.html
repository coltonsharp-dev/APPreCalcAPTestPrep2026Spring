3A.1

Radian Measure — The Definition

Topic 3.2 · 🔥 5/10 Wrong
What a Radian Actually Is

A radian is not just a unit — it's a ratio. Specifically, a radian is the angle whose arc length equals the radius of the circle.

The Definition (This Is What the AP Tests)
One radian = the angle θ such that the arc it intercepts has length equal to the radius r

More generally:   θ (in radians) = arc length sradius r = sr

Or:   s = r·θ   ← arc length formula
💡 Why 2π radians = 360°: A full circle has circumference 2πr. Dividing by r: 2πr/r = 2π. So a full circle = 2π radians. This isn't a conversion fact to memorize — it follows directly from the definition s/r = θ where s = full circumference = 2πr.
Degrees ↔ Radians Conversion
Conversion Formulas
Degrees → Radians:   θrad = θdeg × π180
Radians → Degrees:   θdeg = θrad × 180π

Key pairs to know cold:
0° = 0  |  30° = π/6  |  45° = π/4  |  60° = π/3  |  90° = π/2
120° = 2π/3  |  135° = 3π/4  |  150° = 5π/6  |  180° = π
210° = 7π/6  |  225° = 5π/4  |  240° = 4π/3  |  270° = 3π/2
300° = 5π/3  |  315° = 7π/4  |  330° = 11π/6  |  360° = 2π
⚠️ AP radian definition question pattern (Q16 type): "An angle of θ radians is formed in a circle with radius r. The arc length intercepted is s. Which of the following must be true?" Answer: s = rθ. The AP tests whether you know the definition, not just the conversion table.
Practice · Radian Definition & Conversion
Problem — Definition Type (Q16 Type, 5/10 Wrong)
A circle has radius 5 cm. An angle θ intercepts an arc of length 10 cm.
(a) Find θ in radians.
(b) Convert θ to degrees.
(c) If the radius were doubled to 10 cm but θ stayed the same, what would the new arc length be?
1
(a) Use the definition: θ = s/r
θ = s/r = 10/5 = 2 radians
→ This IS the definition. Radian = arc length divided by radius. No formula needed beyond knowing the definition.
2
(b) Convert 2 radians to degrees
2 × (180/π) = 360/π ≈ 114.6°
3
(c) Double the radius, same angle
s = r·θ = 10 · 2 = 20 cm
→ Arc length scales proportionally with radius. Doubling r doubles s for the same angle.
Problem
Convert: (a) 5π/4 to degrees   (b) 210° to radians   (c) Find the arc length on a circle of radius 6 cm intercepted by an angle of π/3 radians.
(a) 5π/4 × (180/π) = ?
(b) 210° × (π/180) = ?
(c) Arc length s = r·θ with r=6, θ=π/3
(a) 5π/4 × 180/π = 5×45 = 225°
(b) 210 × π/180 = 7π/6 = 7π/6 radians
(c) s = 6·(π/3) = 2π cm ≈ 6.28 cm
Blind — Definition-Based (AP MCQ style)
(a) A wheel of radius 4 feet rotates through an angle of 3π/2 radians. How far does a point on the rim travel?

(b) An angle θ intercepts an arc of length 12π cm on a circle of circumference 36π cm. Find θ in radians and degrees.

(c) True or false: "A 2-radian angle is larger than a 90° angle." Justify.
(a) s = r·θ = 4·(3π/2) = 6π feet ≈ 18.85 feet

(b) Circumference = 36π, so r = 18 (since C=2πr → r=18). θ = s/r = 12π/18 = 2π/3 radians = 120°

(c) True. 90° = π/2 ≈ 1.571 radians. 2 radians > π/2 ≈ 1.571. So a 2-radian angle is slightly larger than a right angle.
3A.2

The Unit Circle

Topic 3.3–3.4
Why the Unit Circle Has Radius 1

The unit circle is a circle centered at the origin with radius r = 1. Because r = 1, the arc length formula s = rθ becomes s = θ, meaning arc length in radians equals the angle itself. This is also why trig functions become so clean on the unit circle: since r = 1, the coordinates directly equal sin and cos.

The Definitions on the Unit Circle
Trig Functions from Unit Circle Coordinates
A point P = (x, y) on the unit circle at angle θ satisfies:

cos(θ) = x-coordinate    sin(θ) = y-coordinate

tan(θ) = yx = sin θcos θ

csc(θ) = 1sin θ   sec(θ) = 1cos θ   cot(θ) = cos θsin θ
The Complete Unit Circle — All 16 Standard Angles
Each point is (cos θ, sin θ). Blue = QI, Purple = QII, Red = QIII, Orange = QIV. Gold = axes.
Memory Strategy — The Three Special Triangles

The unit circle coordinates come from two special right triangles. Know these and you can reconstruct the entire unit circle.

30-60-90
Sides: 1, √3, 2
sin 30°=1/2, cos 30°=√3/2
sin 60°=√3/2, cos 60°=1/2
45-45-90
Sides: 1, 1, √2
sin 45°=√2/2, cos 45°=√2/2
tan 45°=1
Axes
0°: (1,0)
90°: (0,1)
180°: (−1,0)
270°: (0,−1)
Practice · Unit Circle Values
Problem
Find the exact value of each without a calculator:
(a) cos(2π/3)   (b) sin(5π/4)   (c) tan(7π/6)   (d) cos(−π/6)
1
(a) cos(2π/3) — Quadrant II, reference angle π/3
2π/3 is in QII (between π/2 and π). Reference angle = π − 2π/3 = π/3.
cos(π/3) = 1/2. In QII, cosine is negative.
cos(2π/3) = −1/2
2
(b) sin(5π/4) — Quadrant III, reference angle π/4
5π/4 is in QIII (between π and 3π/2). Reference angle = 5π/4 − π = π/4.
sin(π/4) = √2/2. In QIII, sine is negative.
sin(5π/4) = −√2/2
3
(c) tan(7π/6) — Quadrant III
Reference angle = 7π/6 − π = π/6. tan(π/6) = 1/√3 = √3/3. In QIII, tan is positive (sin and cos both negative, so negative/negative = positive).
tan(7π/6) = √3/3
4
(d) cos(−π/6) — negative angle means clockwise
−π/6 is the same as going clockwise from 0 by π/6, landing in QIV where cosine is positive.
cos(−π/6) = cos(π/6) = √3/2
→ Cosine is an even function: cos(−θ) = cos(θ). So cos(−π/6) = cos(π/6).
Problem
Without a calculator, evaluate: (a) sin(3π/2)   (b) tan(π/4)   (c) sec(π/3)   (d) csc(5π/6)
(a) 3π/2 is on a coordinate axis. What point is at 270° on the unit circle?
(b) tan(π/4): use the triangle or coordinates at 45°. cos=sin=√2/2. tan=sin/cos=?
(c) sec(π/3) = 1/cos(π/3). What is cos(60°)?
(d) csc(5π/6) = 1/sin(5π/6). Reference angle = π−5π/6 = π/6, QII so sin positive.
(a) Point at 3π/2 is (0,−1) → sin(3π/2) = −1
(b) tan(π/4) = 1 → 1
(c) cos(π/3)=1/2 → sec(π/3) = 1/(1/2) = 2
(d) sin(5π/6)=sin(π/6)=1/2 → csc(5π/6) = 2
Blind — All Six Functions
θ is in Quadrant III and cos θ = −4/5. Find sin θ, tan θ, csc θ, sec θ, and cot θ.
QII: cos = x = −4/5. Using x²+y²=1: (4/5)²+y²=1 → y²=9/25 → y=±3/5. In QIII, y<0 → sin θ = y = −3/5

tan θ = sin/cos = (−3/5)/(−4/5) = 3/4 (positive in QIII ✓)
csc θ = 1/sin = 1/(−3/5) = −5/3
sec θ = 1/cos = 1/(−4/5) = −5/4
cot θ = cos/sin = (−4/5)/(−3/5) = 4/3
3A.3

Reference Angles & Quadrant Signs

Topics 3.4–3.5
Reference Angles — The Shortcut to Any Exact Value

The reference angle is the positive acute angle between the terminal side and the x-axis. It's always between 0 and π/2. Every non-axis angle has a reference angle, and the trig values at the original angle equal the trig values at the reference angle — with the sign adjusted for the quadrant.

Reference Angle by Quadrant
QI (0 to π/2):    ref = θ itself
QII (π/2 to π):   ref = π − θ
QIII (π to 3π/2): ref = θ − π
QIV (3π/2 to 2π): ref = 2π − θ

Then: |trig(θ)| = trig(ref angle). Adjust sign based on quadrant.
Sign of Each Trig Function by Quadrant — ASTC

0 < θ < π/2

Cos, Tan negative
π/2 < θ < π
QIII
Tan positive
Sin, Cos negative
π < θ < 3π/2
QIV
Cos positive
Sin, Tan negative
3π/2 < θ < 2π
The ASTC Memory Device
All Students Take Calculus

QI = All positive   QII = Sine positive   QIII = Tangent positive   QIV = Cosine positive

Reciprocals follow the same sign as their base: csc follows sin, sec follows cos, cot follows tan.

Practice · Reference Angles & Signs
Problem — Full Reference Angle Process
Find the exact value of tan(11π/6) using the reference angle method. Show every step.
1
Identify the quadrant
11π/6 ≈ 5.76 radians. 3π/2 ≈ 4.71 and 2π ≈ 6.28. Since 4.71 < 5.76 < 6.28 → QIV
2
Find the reference angle
QIV: ref = 2π − θ = 2π − 11π/6 = 12π/6 − 11π/6 = π/6
3
Evaluate at reference angle
tan(π/6) = 1/√3 = √3/3
4
Apply the quadrant sign
In QIV, tangent is negative (ASTC: only Cosine positive in QIV)
tan(11π/6) = −√3/3
Problem
Find the exact value using reference angles: (a) sin(4π/3)   (b) cos(5π/4)   (c) tan(5π/3)
(a) 4π/3: which quadrant? Ref angle = 4π/3 − π = ? Sign of sin in that quadrant?
(b) 5π/4: which quadrant? Ref angle = 5π/4 − π = ? Sign of cos?
(c) 5π/3: which quadrant? Ref = 2π−5π/3 = ? Sign of tan?
(a) QIII. Ref=π/3. sin(π/3)=√3/2. QIII sin is negative → −√3/2
(b) QIII. Ref=π/4. cos(π/4)=√2/2. QIII cos is negative → −√2/2
(c) QIV. Ref=π/3. tan(π/3)=√3. QIV tan is negative → −√3
Blind — Mixed Exact Values
Evaluate exactly (no calculator):
(a) sin(−4π/3)   (b) cos(17π/6)   (c) tan(−7π/4)   (d) csc(5π/3)
(a) sin(−4π/3): sin is odd → sin(−θ)=−sin(θ) → −sin(4π/3). QIII ref=π/3. sin(π/3)=√3/2, QIII sin negative → sin(4π/3)=−√3/2. So sin(−4π/3) = −(−√3/2) = √3/2

(b) cos(17π/6): 17π/6 = 2π + 5π/6 → same as cos(5π/6). QII ref=π/6. cos(π/6)=√3/2. QII cos negative → −√3/2

(c) tan(−7π/4): tan is odd → −tan(7π/4). QIV ref=π/4. tan(π/4)=1. QIV tan negative → tan(7π/4)=−1. So tan(−7π/4) = −(−1) = 1

(d) csc(5π/3) = 1/sin(5π/3). QIV ref=π/3. sin(π/3)=√3/2. QIV sin negative → sin(5π/3)=−√3/2. csc = 1/(−√3/2) = −2/√3 = −2√3/3
