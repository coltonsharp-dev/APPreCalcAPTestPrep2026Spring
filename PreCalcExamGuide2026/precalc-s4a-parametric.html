4A.1

What Parametric Functions Are

Topic 4.1
The Core Idea

A parametric function describes a curve where both x and y are defined separately as functions of a third variable t (the parameter). Instead of y = f(x), you have:

Parametric Form
x(t) = [some function of t]
y(t) = [some function of t]

As t increases through its domain, the point (x(t), y(t)) traces a path in the xy-plane with a specific direction.

t is often time. The parametric equations describe WHERE something is (via x and y) and implicitly WHEN it's there (via t).
💡 Why parametric? Some curves can't be written as y=f(x) (like circles, which fail the vertical line test). Parametric equations handle any path — including ones that loop, go backwards, or cross themselves. The parameter t adds the crucial dimension of direction and timing.
Plotting a Parametric Curve — Build a Table

Choose values of t, compute x(t) and y(t) for each, plot the points (x, y), and connect them in the order of increasing t. Add arrows showing direction of travel.

Example — Plot x(t) = t², y(t) = t − 1 for t ∈ [−2, 2]
tx(t) = t²y(t) = t−1Point (x,y)
−24−3(4, −3)
−11−2(1, −2)
00−1(0, −1)
110(1, 0)
241(4, 1)
Connect in t-order: (4,−3)→(1,−2)→(0,−1)→(1,0)→(4,1). The curve looks like a sideways parabola with direction arrows pointing upward (as t increases, y increases).
Finding Parameter Values — Given a Point on the Curve

If given a point (a, b), find all values of t such that x(t) = a AND y(t) = b simultaneously. Both equations must be satisfied by the same t.

Strategy — Solve Both Equations, Check for Agreement
Example: x(t) = t², y(t) = t − 1. Is the point (4, 1) on the curve?

From x(t)=4: t²=4 → t=2 or t=−2
From y(t)=1: t−1=1 → t=2

t=2 satisfies BOTH → yes, (4,1) is on the curve at t=2.
t=−2 satisfies x but not y (y(−2)=−3≠1) → that t is rejected.
❌ The Q1 miss pattern: Students solve only x(t)=a and stop, without checking that the same t also satisfies y(t)=b. You must verify BOTH equations give the same t value.
Distance Between Two Points on a Parametric Curve

The distance between two points on a parametric curve at t=t₁ and t=t₂ is just the Euclidean distance between those two points:

Distance Formula for Parametric Points
d = √[(x(t₂) − x(t₁))² + (y(t₂) − y(t₁))²]

This is the straight-line distance between the two positions — NOT the arc length along the curve.

Blind — Robot Position (Q5 Type)
A robot's position at time t seconds is given by x(t) = 3cos(πt) and y(t) = 3sin(πt), for t ≥ 0.

(a) Find the robot's position at t=0, t=0.5, t=1, t=1.5.
(b) What shape does the robot trace? What is the period?
(c) At what time does the robot first return to its starting position?
(d) How far is the robot from the origin at all times?
(a) t=0: (3,0). t=0.5: (3cos(π/2),3sin(π/2))=(0,3). t=1: (−3,0). t=1.5: (0,−3).

(b) The robot traces a circle of radius 3 centered at the origin. Period = 2π/π = 2 seconds.

(c) Robot first returns to (3,0) at t=2 seconds (one full period).

(d) Distance from origin = √(x²+y²) = √(9cos²(πt)+9sin²(πt)) = √(9·1) = 3 units at all times.
4A.2

Parametric Motion Analysis

Topic 4.2 · Projectile, Intercepts, Extrema
x-Intercepts and y-Intercepts of a Parametric Curve

To find intercepts of a parametric curve, find the t-values that make one coordinate zero — but verify the curve actually exists at that t within its domain.

Finding Intercepts
x-intercepts: Set y(t) = 0, solve for t. Compute x(t) at those t values.
y-intercepts: Set x(t) = 0, solve for t. Compute y(t) at those t values.

Example: x(t) = t−1, y(t) = t²−4
x-int: y(t)=0 → t=±2. Points: x(2)=1 and x(−2)=−3 → x-intercepts at (1,0) and (−3,0)
y-int: x(t)=0 → t=1. Point: y(1)=1−4=−3 → y-intercept at (0,−3)
Local Extrema of x(t) and y(t)

The x and y coordinates each have their own extrema as functions of t. To find where x(t) has a local max/min, treat x(t) as a function of t and use the same process as for any function — find where the rate of change of x changes sign. Similarly for y(t).

Finding Extrema of Each Coordinate
To find where x(t) is maximized: find the t value where x(t) has a peak (x(t) increases before, decreases after).

To find where y(t) is maximized: find the t value where y(t) has a peak.

The x and y extrema don't necessarily occur at the same t — they are independent. A point on the curve where x is maximum isn't necessarily where y is maximum.
Projectile Motion — The Standard Parametric Model
Projectile Motion Model
x(t) = v₀ cos(θ) · t   ← horizontal motion (constant speed)
y(t) = v₀ sin(θ) · t − ½g·t² + h₀  ← vertical (affected by gravity)

where v₀ = initial speed, θ = launch angle, g = 9.8 m/s² or 32 ft/s², h₀ = initial height

To find when it lands: set y(t) = 0 (or ground level) and solve for t.
Maximum height: find t that maximizes y(t).
Horizontal range: x at the landing t value.

4A.3

Increasing, Decreasing & Rates of Change

Topic 4.3
When Is a Parametric Curve Increasing or Decreasing?

A parametric curve is described as "increasing" or "decreasing" based on how y changes as x increases — not based on t directly. But since both x and y change with t, you need to analyze the relationship carefully.

The Key Framework
The parametric curve is increasing (y increases as x increases) when:
    x(t) increasing AND y(t) increasing at the same time (both rates positive)
    OR x(t) decreasing AND y(t) decreasing at the same time (both rates negative)

The parametric curve is decreasing (y decreases as x increases) when:
    x(t) increasing AND y(t) decreasing (opposite signs)
    OR x(t) decreasing AND y(t) increasing (opposite signs)
⚠️ Increasing in t ≠ increasing as a curve. If x(t) is decreasing as t increases, but y(t) is also decreasing as t increases, the curve is going "backwards-left and down" — which means as x increases, y actually increases (since x is going backwards). Think carefully about direction.
Rate of Change of y with Respect to x

The rate at which y changes per unit change in x — the slope of the curve at a point — depends on both rates:

Rate of Change dy/dx from Parametric Equations
dydx = rate y changes with trate x changes with t = Δy/ΔtΔx/Δt

From a table: slope ≈ y(t₂) − y(t₁)x(t₂) − x(t₁)

The curve is increasing where dy/dx > 0 (y and x change in same direction)
The curve is decreasing where dy/dx < 0 (y and x change in opposite directions)
Traversing Similar Curves — The Q11 Concept

Two different parametric equations can trace the same geometric curve but at different speeds, in different directions, or over different time intervals. The orientation (direction of traversal) and speed can differ even when the underlying curve looks the same.

Example — Same Curve, Different Traversal
Curve A: x(t) = t, y(t) = t² for t ∈ [0, 2]
Curve B: x(t) = 2−t, y(t) = (2−t)² for t ∈ [0, 2]

Both trace the parabola y = x² for x ∈ [0, 2].
Curve A: goes LEFT TO RIGHT (x increases as t increases)
Curve B: goes RIGHT TO LEFT (x decreases as t increases)

Same shape — opposite direction.
