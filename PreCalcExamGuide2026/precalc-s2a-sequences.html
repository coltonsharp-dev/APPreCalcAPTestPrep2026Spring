2A.1

Arithmetic & Geometric Sequences

Topic 2.1–2.2
The Two Types — Side by Side
Arithmetic
Add a constant
Pattern:Add d each term
Common diff:d = aₙ − aₙ₋₁
nth term:aₙ = a₁ + (n−1)d
Looks like:Linear function
Example:3, 7, 11, 15... (d=4)
Finite differences:Constant 1st diff
Geometric
Multiply a constant
Pattern:Multiply r each term
Common ratio:r = aₙ / aₙ₋₁
nth term:aₙ = a₁ · rⁿ⁻¹
Looks like:Exponential function
Example:2, 6, 18, 54... (r=3)
Finite differences:Constant ratio
Identifying the Type from a Table

Given a table with equally spaced inputs, check whether the differences or ratios of outputs are constant:

Detection Method
If output differences are constant → arithmetic (linear behavior)
If output ratios are constant → geometric (exponential behavior)

Table: x = 0,1,2,3 and f(x) = 4, 12, 36, 108
Ratios: 12/4 = 3, 36/12 = 3, 108/36 = 3 → constant ratio → geometric/exponential
⚠️ AP trap: Tables with equally spaced inputs. If inputs are not equally spaced, the constant difference / ratio test doesn't apply directly — you need to account for the step size in the input.
Finding a Specific Term
Geometric nth Term — Full Process
Sequence: 5, 15, 45, 135, ...
a₁ = 5, r = 15/5 = 3
aₙ = 5 · 3ⁿ⁻¹

Find a₇: a₇ = 5 · 3⁶ = 5 · 729 = 3645

The exponent is n−1, NOT n. First term uses n=1, so exponent = 1−1 = 0, giving a₁·r⁰ = a₁ ✓
❌ Off-by-one error: Writing aₙ = a₁ · rⁿ instead of a₁ · rⁿ⁻¹. This is the most common geometric sequence mistake. Check: plug in n=1 to verify you get a₁.
Repeated Behavior — Sequence Cycles

Some sequences have repeating patterns. To find the value of the nth term, find the remainder when n is divided by the cycle length. The remainder tells you which position in the cycle you're at.

Repeating Sequence Example
Sequence: 1, 3, 9, 1, 3, 9, 1, 3, 9, ... (cycle length = 3)

Find a₂₀: 20 ÷ 3 = 6 remainder 2 → a₂₀ = 2nd element = 3

Find a₃₀: 30 ÷ 3 = 10 remainder 0 → remainder 0 means the last element → 9

Remainder 0 = last element in the cycle. Remainder 1 = first element. Remainder 2 = second element. Etc.

Problem — Geometric Sequence
A bouncing ball reaches 80% of its previous height with each bounce. It is dropped from 200 cm.
(a) Write the sequence of heights for the first 4 bounces.
(b) Find the height after the 8th bounce.
(c) Is this arithmetic or geometric? How do you know?
1
(a) First four terms — a₁ = 200, r = 0.8
a₁ = 200
a₂ = 200 × 0.8 = 160
a₃ = 160 × 0.8 = 128
a₄ = 128 × 0.8 = 102.4 cm
2
(b) 8th bounce — use aₙ = a₁ · rⁿ⁻¹
a₈ = 200 · (0.8)⁷ = 200 · 0.2097... ≈ 41.9 cm
→ Exponent is n−1 = 8−1 = 7. Don't use n=8 as the exponent.
3
(c) Arithmetic or geometric?
Geometric — each term is multiplied by a constant ratio (0.8). The differences between terms are NOT constant (200→160 is −40, 160→128 is −32). Only the ratios are constant.

2A.2

Exponential Functions

Topics 2.3–2.5
The General Form & Parameters
Standard Form
f(x) = a · bx

a = initial value (the y-intercept, value when x=0). If a > 0: starts positive. If a < 0: starts negative.
b = base (growth or decay factor). b > 0, b ≠ 1
  If b > 1: exponential growth (multiplying by something > 1)
  If 0 < b < 1: exponential decay (multiplying by something < 1)
Percent Change → Exponential Base

When something grows or decays by a constant percent, the base b is built from that percent:

Percent to Base Conversion
r% growth each period: base b = 1 + r100   Example: 5% growth → b = 1.05

r% decay each period: base b = 1 − r100    Example: 8% decay → b = 0.92

Full model: f(t) = a · (1 ± r)ᵗ  ← t = number of periods elapsed
💡 Reverse reading: If you're given b = 1.12, that means 12% growth per period. If b = 0.85, that means 15% decay per period. Subtract b from 1 for decay: 1 − 0.85 = 0.15 = 15% decay.
End Behavior — The Watch Zone from Your Data

Exponential end behavior depends on whether b > 1 or 0 < b < 1, and whether a is positive or negative.

FunctionAs x → +∞As x → −∞HA
f(x) = a·bˣ, b>1, a>0f → +∞ (grows)f → 0⁺y = 0
f(x) = a·bˣ, 0<b<1, a>0f → 0⁺ (decays)f → +∞y = 0
f(x) = a·bˣ + cDepends on bDepends on by = c (vertical shift)
⚠️ Key insight: All exponential functions approach the horizontal asymptote y=0 (or y=c if shifted) on one side. The side depends on the base. Exponentials NEVER touch the asymptote — they approach but stay above (or below if a < 0).
Practice · Exponential Functions & Modeling
Problem — Modeling (2.5 Type)
A population of bacteria starts at 500 and triples every 4 hours.
(a) Write an exponential model for P(t), where t is hours.
(b) Find P(12).
(c) What is the horizontal asymptote, and what does it represent in context?
(d) What is the percent growth rate per hour?
1
(a) Build the model — triples every 4 hours
a = 500 (initial population)
Triples every 4 hours → after t hours: P(t) = 500 · 3^(t/4)
Or equivalently: P(t) = 500 · (3^(1/4))ᵗ
→ t/4 counts how many 4-hour periods have passed. Each complete 4-hour block multiplies by 3.
2
(b) P(12)
P(12) = 500 · 3^(12/4) = 500 · 3³ = 500 · 27 = 13,500 bacteria
3
(c) Horizontal asymptote
HA at y = 0. As t → −∞, P → 0. In context: "approaching zero bacteria" (going back in time before the bacteria existed). As t → +∞, P grows without bound — no asymptote on the growth side.
4
(d) Percent growth per hour
Base per hour = 3^(1/4) ≈ 1.316
Percent growth per hour = ≈ 31.6% per hour
→ Subtract 1 from the base and convert to percent: 1.316 − 1 = 0.316 = 31.6%
Problem
A car's value decreases by 12% each year. It was worth $24,000 new.
(a) Write the exponential model V(t).
(b) Find V(5). Round to the nearest dollar.
(c) What is the annual decay factor and what percent of value is lost each year?
(d) Describe the end behavior as t → +∞.
(a) 12% decay each year → base = 1 − 0.12 = ? Write V(t) = a · bᵗ
(b) V(5) = 24000 · (0.88)⁵ = ?
(c) State the decay factor and percent lost annually
(d) End behavior as t → +∞
(a) V(t) = 24000 · (0.88)ᵗ
(b) V(5) = 24000 · (0.88)⁵ = 24000 · 0.5277 ≈ $12,666
(c) Decay factor = 0.88. Percent lost = 12% per year.
(d) As t → +∞, V(t) → 0⁺ (asymptotically approaches $0 but never reaches it). In context: the car's value approaches but never quite reaches zero.

Exponent Manipulation

Topic 2.4 · 🔥 Sub-topic Watch
All the Rules — In One Place
Product Rule
bᵐ · bⁿ = bᵐ⁺ⁿ
Same base → add exponents
3² · 3⁴ = 3⁶ = 729
Quotient Rule
bᵐ / bⁿ = bᵐ⁻ⁿ
Same base → subtract exponents
5⁷ / 5³ = 5⁴ = 625
Power Rule
(bᵐ)ⁿ = bᵐⁿ
Power of a power → multiply
(2³)⁴ = 2¹² = 4096
Zero Exponent
b⁰ = 1 (b ≠ 0)
Any nonzero base to the 0th power equals 1
7⁰ = 1, (−5)⁰ = 1
Negative Exponent
b⁻ⁿ = 1/bⁿ
Negative exponent → reciprocal
2⁻³ = 1/2³ = 1/8
Fractional Exponent
b^(m/n) = (ⁿ√b)ᵐ
n = root, m = power
8^(2/3) = (∛8)² = 2² = 4
Fractional = Root
b^(1/n) = ⁿ√b
1/n exponent = nth root
16^(1/4) = ⁴√16 = 2
Power of Product
(ab)ⁿ = aⁿbⁿ
Distribute to each factor
(2x)³ = 8x³
Fractional Exponents — Full Breakdown

This is the most-missed sub-topic. The key is knowing which part of m/n is the root and which is the power:

Fractional Exponent → Root and Power
b^(m/n) = (ⁿ√b)ᵐ = ⁿ√(bᵐ)

Denominator = root  |  Numerator = power

Worked examples:
27^(2/3): root = 3, power = 2 → (∛27)² = 3² = 9
16^(3/4): root = 4, power = 3 → (⁴√16)³ = 2³ = 8
32^(−2/5): negative → reciprocal first → 1/32^(2/5) = 1/(⁵√32)² = 1/2² = 1/4
💡 Order of operations for fractional exponents: Take the root FIRST (if possible), then raise to the power. This keeps the numbers small and manageable. 8^(2/3): take ∛8 = 2 first, then 2² = 4. Much easier than 8² = 64 then ∛64 = 4.
Equivalent Exponential Forms — Rewriting

The AP exam frequently asks you to rewrite an exponential expression in a different equivalent form. The goal is to recognize the same base with a different exponent.

Rewriting Equivalent Forms
2^(3t) = (2³)ᵗ = 8ᵗ     ← group the base differently
9ˣ = (3²)ˣ = 3^(2x)   ← express in terms of prime base
4^(x/2) = (4^(1/2))ˣ = 2ˣ  ← simplify the fractional exponent

These rewrites appear when matching: "Which is equivalent to f(x) = 4·8^(2x)?"

(A) 5 · 64ˣ    (B) 20 · 4^(3x)    (C) 5 · 12ˣ    (D) 5 · 4^x

Blind — Full Simplification Set
Simplify each completely:
(a) (x^(2/3) · x^(1/6))^6    (b) (8x³)^(−2/3)    (c) 4^(x+1) / 2^x    (d) (27/8)^(−2/3)
