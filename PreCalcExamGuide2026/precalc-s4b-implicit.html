4B.1

Parametrically Defined Lines

Topic 4.4
The Parametric Line Form

A line in the plane can be written parametrically by specifying a starting point (x₀, y₀) and a direction vector (a, b). As t varies, the point (x, y) moves along the line.

Parametric Line — Standard Form
x(t) = x₀ + a·t
y(t) = y₀ + b·b·t

where (x₀, y₀) is a point on the line at t=0, and (a, b) is the direction vector

Slope of the line = ba (rise over run from the direction vector)
Converting Parametric Line ↔ Rectangular Form

To eliminate t and get y = mx + c: solve one equation for t, substitute into the other.

Parametric → Rectangular: Eliminate t
x(t) = 2 + 3t    y(t) = 1 − 2t

From x: 3t = x−2 → t = x−23

Substitute into y: y = 1 − 2·x−23 = 1 − 2x−43 = 3−2x+43 = 7−2x3

Or: y = −23x + 73
Rectangular → Parametric: Introduce t
y = 3x − 5 → Let x(t) = t, then y(t) = 3t − 5
Or: Let x(t) = t + 2, then y(t) = 3(t+2)−5 = 3t+1 (different parameterization, same line)

There are infinitely many valid parameterizations. The AP just asks you to find one valid form or match a given one.

4B.2

Parametrically Defined Circles

Topic 4.4 · Q14 Watch
The Standard Parametric Circle

A circle of radius r centered at (h, k) has the parametric form:

Parametric Circle — All Four Variations
x(t) = r·cos(t) + h
y(t) = r·sin(t) + k

where r = radius, (h, k) = center, t ∈ [0, 2π]

Identifying from the equation:
x(t) = a·cos(t) + h: the coefficient of cos is the radius. The constant is the x-center.
y(t) = a·sin(t) + k: the coefficient of sin is the radius. The constant is the y-center.
Transformations of Parametric Circles — The Q14 Type

When a parametric circle is transformed, each parameter in (r, h, k) changes in a specific way. Recognizing which transformation was applied is the key skill.

Transformation AppliedEffect on Parametric EquationsEffect on Circle
Shift right by cx(t) = r·cos(t) + (h+c)Center moves right, radius unchanged
Shift up by cy(t) = r·sin(t) + (k+c)Center moves up, radius unchanged
Scale radius by factor sx(t) = (r·s)·cos(t) + h, y(t) = (r·s)·sin(t) + kRadius changes, center unchanged
Reflect over x-axisy(t) = −r·sin(t) − k (negate y)Circle flips — center goes to (h, −k)
⚠️ The Q14 pattern: You're given x(t)=a·cos(t)+h, y(t)=b·sin(t)+k and asked what happens when a transformation is applied. Read the modified constants carefully — if a changes but h doesn't, it's a radius change. If h changes but a doesn't, it's a center shift.
Converting Parametric Circle → Rectangular Form

Use the Pythagorean identity cos²t + sin²t = 1 to eliminate t:

Elimination via Pythagorean Identity
x(t) = 3cos(t) + 2  →  cos(t) = (x−2)/3
y(t) = 3sin(t) − 1  →  sin(t) = (y+1)/3

cos²t + sin²t = 1:  ((x−2)/3)² + ((y+1)/3)² = 1

→ (x−2)² + (y+1)² = 9  ← circle: center (2,−1), radius 3

4B.3

Implicitly Defined Relations

Topic 4.5 · Q15–Q17
What Implicitly Defined Means

An implicitly defined relation is an equation in x and y where y is NOT isolated — both variables appear together. The set of all (x, y) pairs satisfying the equation forms a curve in the plane, but that curve may not pass the vertical line test.

Examples of Implicit Relations
x² + y² = 25        ← circle (y not isolated)
x² + 4y² = 16      ← ellipse
xy = 12            ← hyperbola
y² = x + 3         ← sideways parabola
x³ + y³ = 3xy     ← folium (complex curve)
Reading Graphs of Implicit Relations

To determine if a point (a, b) lies on an implicitly defined curve, substitute x=a and y=b into the equation and check if it's satisfied.

Point on Curve — Check by Substitution
Is (3, 4) on x² + y² = 25?
3² + 4² = 9 + 16 = 25 = 25 ✓ → yes, (3,4) is on the circle

Is (2, 3) on x² + y² = 25?
4 + 9 = 13 ≠ 25 → no, (2,3) is not on the circle
Finding Intersection Points — The Q15 Type

To find where two implicit curves intersect, find the (x, y) pairs that satisfy both equations simultaneously. This is a system of equations.

Intersection Point Strategy
Curve 1: x² + y² = 25
Curve 2: y = x − 1

Substitute Curve 2 into Curve 1:
x² + (x−1)² = 25
x² + x² − 2x + 1 = 25
2x² − 2x − 24 = 0
x² − x − 12 = 0
(x−4)(x+3) = 0 → x = 4 or x = −3

y = x−1: y(4) = 3, y(−3) = −4
Intersection points: (4, 3) and (−3, −4)
Rate of Change on an Implicit Curve — The Q17 Type

The rate of change of y with respect to x along an implicit curve is found differently from explicit functions — but at the AP Pre-Calc level, it's approximated numerically from nearby points on the curve, or from the slope of a tangent line at a given point.

Approximating Rate at a Point on an Implicit Curve
Given: x² + y² = 25, find the approximate rate of change at (3, 4).

Find a nearby point on the same curve: at x=3.1: 3.1²+y²=25 → y²=15.39 → y≈3.923
Rate ≈ ΔyΔx = 3.923−43.1−3 = −0.0770.1 ≈ −0.77

The tangent line at (3,4) on this circle has slope approximately −3/4 ≈ −0.75 (by symmetry of circles). Close to our approximation ✓

For the AP Pre-Calc exam, you approximate from nearby points or are given values — full implicit differentiation isn't required.
