3B.1

The Four Parameters — A, B, C, D

Topics 3.6–3.7
The General Form
Standard Sinusoidal Form
f(x) = A · sin(B(x − C)) + D

    or

f(x) = A · cos(B(x − C)) + D

All four parameters can be positive or negative. Each controls a specific feature of the graph.
A
Amplitude
|A| = half the vertical range
A = (max − min) / 2
If A is negative: graph reflects over midline (flipped vertically)
Max value = D + |A|, Min = D − |A|
B
Angular Frequency → Period
Period = 2π / |B|
Larger B → shorter period (faster oscillation)
B = 2π / period
Read period from graph: peak to next peak (or any point to same point one cycle later)
C
Phase Shift (Horizontal)
Shifts graph RIGHT by C units
f(x) = sin(B(x − C)): shifts right C
f(x) = sin(Bx − D): phase shift = D/B
Negative C: shifts LEFT
D
Midline (Vertical Shift)
D = (max + min) / 2
The horizontal line the wave oscillates around
Positive D: shifts graph UP
This is the average value of the function
⚠️ Period formula trap: The period is 2π/|B|, NOT 2π·B. If B=2, the period is π (shorter, not longer). Larger B = more oscillations per unit = shorter period. This is the #1 sinusoidal error.
Red dots = max/min. Green dots = midline crossings. Red arrow = amplitude A. Blue bracket = period. Green dashed = midline y=D.
Reading All Four Parameters from a Graph — Step by Step
1
Find D (midline): D = (max + min) / 2. Draw the horizontal line halfway between the peaks and troughs.
2
Find A (amplitude): A = (max − min) / 2. Distance from midline to a peak. Check if graph is flipped (negative A).
3
Find B (from period): read the period from graph (peak to peak), then B = 2π / period.
4
Find C (phase shift): where does the pattern start relative to the parent? If using sin: find where the function crosses the midline going upward — that x is C. If using cos: find where the maximum occurs — that x is C.
Practice · Reading Parameters from a Graph
Problem — Full Parameter Extraction
A sinusoidal graph has: maximum value of 9 at x = 1, minimum value of 3 at x = 5, and passes through the midline going upward at x = −1.

Write the equation in both sine and cosine form.
1
Find D (midline)
D = (max + min)/2 = (9+3)/2 = 6
2
Find A (amplitude)
A = (max − min)/2 = (9−3)/2 = 3
3
Find the period and B
From midline up at x=−1 to max at x=1: that's a quarter period. Quarter period = 1−(−1) = 2. So full period = 8.
B = 2π/8 = π/4
→ Alternatively: max at x=1 to min at x=5 is half a period: half = 5−1=4, so full period = 8.
4
Write sine form — C is where midline is crossed going upward
Midline crossed going upward at x=−1 → C = −1
f(x) = 3·sin(π4(x−(−1))) + 6 = 3 sin(π(x+1)/4) + 6
5
Write cosine form — C is where the maximum occurs
Maximum at x=1 → C = 1
f(x) = 3 cos(π(x−1)/4) + 6
Problem
A sinusoidal function has maximum 12, minimum −4, period 6, and the first maximum occurs at x = 2. Write the equation using cosine.
Step 1 — Find D and A
Step 2 — Find B from the period (period = 6)
Step 3 — For cosine: C = x-value of first maximum = 2. Write f(x) = A·cos(B(x−C)) + D
D = (12+(−4))/2 = 4. A = (12−(−4))/2 = 8. B = 2π/6 = π/3. C = 2 (first max at x=2).

f(x) = 8·cos(π(x−2)/3) + 4

Verify: f(2) = 8cos(0)+4 = 8+4 = 12 ✓ (max value)
Blind — Read from Equation
For f(x) = −5·sin(2x − π/3) + 1, find: (a) amplitude, (b) period, (c) midline, (d) phase shift, (e) maximum and minimum values, (f) is the graph reflected (flipped)?
(a) Amplitude = |−5| = 5
(b) B=2 → period = 2π/2 = π
(c) Midline: y = 1
(d) Rewrite: f = −5sin(2(x − π/6)) + 1. Phase shift = π/6 to the right
(e) Max = D + |A| = 1+5 = 6. Min = D−|A| = 1−5 = −4
(f) Yes, reflected — the negative A flips the graph over the midline. Starts at midline going down (instead of up for positive sin).

Sinusoidal Modeling — Real-World Contexts

Topic 3.7 · The Most Common AP Context
Building the Model from Context

The AP provides real-world data (tide heights, daylight hours, temperatures, rotating objects) and asks you to build or interpret a sinusoidal model. The key is extracting the four parameters from the context language — not from a graph.

Context → Parameter Mapping
"Average value / rest position" → D (midline)
"Maximum deviation from average / amplitude of oscillation" → A
"Period / cycle length / time for one complete cycle" → use to find B = 2π/period
"Start position / initial condition" → use to find C (phase shift)

Practice · Sinusoidal Modeling
Problem — Tide Modeling (AP Style)
The height of the tide at a harbor can be modeled as a sinusoidal function of time. The tide reaches a maximum height of 11 feet at 2:00 AM and a minimum height of 1 foot at 8:00 AM. Time t is measured in hours since midnight.

(a) Find D, A, and the period.
(b) Write the equation using cosine.
(c) Find the tide height at noon (t=12).
(d) At what times in the first 24 hours does the tide equal 9 feet?
1
(a) D, A, and period
D = (11+1)/2 = 6 feet (average tide)
A = (11−1)/2 = 5 feet (amplitude)
Max at t=2, min at t=8: half period = 8−2 = 6 hours → period = 12 hours
B = 2π/12 = π/6
2
(b) Cosine equation — max at t=2
Cosine: C = time of maximum = 2
h(t) = 5 cos(π(t−2)/6) + 6

Verify: h(2) = 5cos(0)+6 = 11 ✓, h(8) = 5cos(π)+6 = −5+6 = 1 ✓
3
(c) Height at t=12
h(12) = 5cos(π(12−2)/6) + 6 = 5cos(10π/6) + 6 = 5cos(5π/3) + 6
cos(5π/3) = 1/2 (QIV, reference π/3)
= 5(1/2) + 6 = 2.5 + 6 = 8.5 feet
4
(d) When does h(t) = 9?
5cos(π(t−2)/6) + 6 = 9
5cos(π(t−2)/6) = 3
cos(π(t−2)/6) = 3/5
π(t−2)/6 = ±arccos(3/5) ≈ ±0.927
t−2 = ±0.927·(6/π) ≈ ±1.769
t ≈ 2 + 1.769 = 3.77 hours (≈3:46 AM)
t ≈ 2 − 1.769 = 0.23 hours (≈12:14 AM)
Next cycle adds 12: t ≈ 15.77 and 12.23
Problem — Ferris Wheel
A Ferris wheel has a diameter of 40 feet and its center is 25 feet above the ground. It makes one complete revolution every 2 minutes. A rider boards at the bottom.
(a) Write h(t) for the rider's height in feet, where t is seconds.
(b) Find the height at t=30 seconds.
(c) When does the rider first reach the top?
Step 1 — Identify D (center height), A (radius), and period
Step 2 — Rider starts at BOTTOM (h = D−A). Use cosine: cos starts at max, so use −cos for bottom start, or use sine offset.
Step 3 — Find h(30). Convert B = 2π/120 = π/60.
Step 4 — First time at top: h = D+A = 45. When does −20cos(πt/60)+25 = 45?
D=25, A=20 (radius), period=120s, B=2π/120=π/60
Starts at bottom (h=5=D−A), use negative cosine:
h(t) = −20cos(πt/60) + 25

h(30) = −20cos(π·30/60)+25 = −20cos(π/2)+25 = −20(0)+25 = 25 feet (at the side, midline height)

First at top: −20cos(πt/60)+25=45 → cos(πt/60)=−1 → πt/60=π → t=60 seconds (half revolution)
Blind — Daylight Hours
The number of daylight hours in a city can be modeled sinusoidally. On June 21 (day 172 of the year), there are 15 hours of daylight. On December 21 (day 355), there are 9 hours. Model D(t) where t is the day of the year, using cosine.

Then predict daylight hours on March 21 (day 80) and state when daylight hours equal 12.
D = (15+9)/2 = 12 hours (midline = average daylight)
A = (15−9)/2 = 3 hours
Period = 365 days (one year), B = 2π/365
Max at t=172 → C = 172 for cosine

D(t) = 3cos(2π(t−172)/365) + 12

March 21 (day 80): D(80) = 3cos(2π(80−172)/365)+12 = 3cos(2π(−92)/365)+12
= 3cos(−1.582)+12 ≈ 3(−0.002)+12 ≈ 12.0 hours (near equinox ✓)

Daylight = 12: 3cos(2π(t−172)/365) = 0 → cos = 0 → 2π(t−172)/365 = π/2 or 3π/2
t−172 = 365/4 ≈ 91.25 → t ≈ 80 (March 21) or t ≈ 263 (September 20)
12 hours on approximately day 80 (March 21) and day 263 (Sept 20)

Trig Equations & Finding Zeros

Topic 3.10 · Watch — 3/10 Wrong
Solving sin(θ) = k or cos(θ) = k

Trig equations have infinitely many solutions because sin and cos repeat every 2π. The AP asks for solutions on a given interval (usually [0, 2π] or one period).

Strategy for sin(θ) = k
Step 1: Find the reference angle: θref = arcsin(|k|)
Step 2: Identify which quadrants have the correct sign for sin
    sin positive → QI and QII
    sin negative → QIII and QIV
Step 3: Write both solutions in [0, 2π]:
    QI: θ = θref
    QII: θ = π − θref
    QIII: θ = π + θref
    QIV: θ = 2π − θref
Strategy for cos(θ) = k
cos positive → QI (θref) and QIV (2π − θref)
cos negative → QII (π − θref) and QIII (π + θref)
Zeros of Sinusoidal Functions

The zeros of f(x) = A·sin(B(x−C)) + D are where f(x) = 0. Set the expression equal to zero and solve. The zeros of the parent sin are at multiples of π; transformed functions have zeros shifted accordingly.

Zeros of sin — General Pattern
sin(θ) = 0 when θ = nπ for any integer n

For f(x) = sin(Bx): zeros at Bx = nπ → x = nπ/B
These are equally spaced, half a period apart.

For f(x) = A·sin(B(x−C))+D, first set D = 0:
If D ≠ 0, zeros require: sin(B(x−C)) = −D/A → solve that equation
