2B.1

What a Logarithm Actually Means

Topic 2.10
The Definition — Both Directions

A logarithm answers the question: "What exponent on the base gives me this number?"

Logarithm Definition — The Translation
logb(x) = y   ↔   by = x

Read as: "base b raised to the power y gives x"
Or: "log base b of x equals y"

Example: log2(8) = 3   ↔   2³ = 8
Example: log10(1000) = 3   ↔   10³ = 1000
💡 The mental model: When you see logb(x), ask: "What power do I raise b to in order to get x?" That answer is the log. log5(125) = ? → "What power do I raise 5 to to get 125?" → 5³ = 125 → answer is 3.
Common Log and Natural Log
Common Log (log₁₀)
log(x) with no base shown
→ base is 10
log(100) = 2  (10² = 100)
log(1) = 0    (10⁰ = 1)
log(0.1) = −1  (10⁻¹ = 0.1)
Natural Log (ln)
ln(x) means loge(x)
→ base is e ≈ 2.718
ln(e) = 1    (e¹ = e)
ln(1) = 0    (e⁰ = 1)
ln(e²) = 2  (e² = e²)
The Inverse Relationship — Logs Undo Exponentials

logb and bˣ are inverse functions. They undo each other in both orders:

Inverse Cancellation Laws
blogb(x) = x    (for x > 0)  ← exponential undoes log
logb(bˣ) = x               ← log undoes exponential

Examples:  10^(log 7) = 7    log(10⁵) = 5    ln(eˣ) = x    e^(ln 3) = 3
⚠️ Q5 type from your exam (4 wrong): "Which of the following is the inverse of f(x) = 3·2ˣ?" Swapping x and y: x = 3·2^y → 2^y = x/3 → y = log₂(x/3). The inverse of an exponential is a logarithm with the same base. The initial multiplier moves inside the log.
Practice · Log Definition & Inverses
Problem
Evaluate without a calculator: (a) log₄(64)   (b) log₃(1/9)   (c) ln(e⁵)   (d) 10^(log 7). Then find the inverse of f(x) = 5·3ˣ.
1
(a) log₄(64) — "What power of 4 gives 64?"
4¹=4, 4²=16, 4³=64 → log₄(64) = 3
2
(b) log₃(1/9) — "What power of 3 gives 1/9?"
3⁻² = 1/9 → log₃(1/9) = −2
→ Fraction → negative exponent. 1/3² = 1/9.
3
(c) ln(e⁵) — log undoes exponential with same base
loge(e⁵) = 5 directly by the cancellation law.
4
(d) 10^(log 7) — exponential undoes log
= 7 directly by the cancellation law.
5
Inverse of f(x) = 5·3ˣ
Swap x and y: x = 5·3^y
Isolate the exponential: 3^y = x/5
Apply log₃: y = log₃(x/5)
f⁻¹(x) = log₃(x/5)
→ Domain of inverse: x > 0 (log only accepts positive input). Range: all reals.
Problem
Find the inverse of each: (a) g(x) = 4ˣ − 3   (b) h(x) = log₂(x + 6)
(a) g(x) = 4ˣ − 3. Step 1: swap x and y. Step 2: isolate the exponential. Step 3: apply log₄.
(b) h(x) = log₂(x+6). Step 1: swap x and y. Step 2: the variable is now in the log — convert to exponential form. Step 3: solve for y.
(a) x = 4^y−3 → 4^y = x+3 → y = log₄(x+3)
g⁻¹(x) = log₄(x+3), domain: x > −3

(b) x = log₂(y+6) → 2^x = y+6 → y = 2^x − 6
h⁻¹(x) = 2^x − 6, domain: all reals
Blind
Simplify completely: (a) log₅(5^(2x+1))   (b) 2^(log₂(x²−3))   (c) e^(3·ln x)   (d) log(10^x / 10²)
(a) log₅(5^(2x+1)) = 2x+1 (log cancels base-matching exponential)

(b) 2^(log₂(x²−3)) = x²−3 (exponential cancels base-matching log)

(c) e^(3·ln x) = e^(ln x³) = x³ (power rule inside, then cancel)

(d) log(10^x / 10²) = log(10^(x−2)) = x−2
2B.2

Logarithm Properties

Topic 2.12
The Four Core Properties
Product Rule
logb(M·N) = logb(M) + logb(N)
log(3x) = log 3 + log x
Multiplication inside → addition outside
Quotient Rule
logb(M/N) = logb(M) − logb(N)
log(x/5) = log x − log 5
Division inside → subtraction outside
Power Rule
logb(Mⁿ) = n·logb(M)
log(x³) = 3·log x
Exponent inside → multiplier outside
Change of Base
logb(x) = log(x)/log(b) = ln(x)/ln(b)
log₅(12) = log(12)/log(5)
Converts any base to calculator-ready form
❌ The most common log property error: log(M + N) ≠ log M + log N. The product rule applies to multiplication inside the log, not addition. log(x + 5) cannot be split. Also: log(M/N) ≠ log M / log N — division outside is change of base, not the quotient rule.
Two Useful Values to Memorize
Always True
logb(1) = 0  ← because b⁰ = 1 for any b
logb(b) = 1  ← because b¹ = b for any b

Examples: log(1) = 0,   ln(1) = 0,   log₇(1) = 0
log(10) = 1,   ln(e) = 1,   log₃(3) = 1
Practice · Log Properties
Problem — Equivalent Form (AP MCQ Style)
Which expression is equivalent to log₂(4x³/y)?
1
Quotient rule — split the fraction
log₂(4x³/y) = log₂(4x³) − log₂(y)
2
Product rule — split the numerator
= log₂(4) + log₂(x³) − log₂(y)
3
Evaluate log₂(4) and apply power rule to log₂(x³)
log₂(4) = log₂(2²) = 2
log₂(x³) = 3·log₂(x)

= 2 + 3·log₂(x) − log₂(y)
= 2 + 3log₂x − log₂y
→ This is the fully expanded form. Work left to right: fraction first, then multiplication, then exponents.
Problem
Write as a single logarithm: 3log x − 2log y + log 5
Step 1 — Apply power rule in reverse: 3log x = log(?), 2log y = log(?)
Step 2 — log A + log B = log(AB), log A − log B = log(A/B). Combine all three terms.
3log x = log x³  |  2log y = log y²

log x³ − log y² + log 5 = log(x³·5/y²)

= log(5x³/y²)
Blind
Given log 2 ≈ 0.301 and log 3 ≈ 0.477, find without a calculator:
(a) log 6   (b) log 18   (c) log(4/9)   (d) log₃(2) using change of base
(a) log 6 = log(2·3) = log 2 + log 3 = 0.301+0.477 = 0.778

(b) log 18 = log(2·3²) = log 2 + 2log 3 = 0.301+2(0.477) = 0.301+0.954 = 1.255

(c) log(4/9) = log 4 − log 9 = 2log 2 − 2log 3 = 2(0.301)−2(0.477) = 0.602−0.954 = −0.352

(d) log₃(2) = log 2 / log 3 = 0.301/0.477 ≈ 0.631
2B.3

Solving Log & Exponential Equations

Topic 2.13
Strategy 1 — Convert Between Forms

The most direct method: rewrite log form as exponential form (or vice versa) and solve.

Log → Exponential (Isolate, Convert, Solve)
log₃(x − 2) = 4
→ 3⁴ = x − 2   (convert: base^answer = input)
→ 81 = x − 2
→ x = 83
Exponential → Log (Isolate, Apply Log)
5^(2x) = 200
→ log(5^(2x)) = log(200)   (apply log to both sides)
→ 2x · log(5) = log(200)  (power rule)
→ x = log(200)2·log(5) ≈ 1.646
Strategy 2 — Same Base Both Sides

If both sides can be written as the same base, set the exponents equal.

Same Base → Equal Exponents
4^(2x−1) = 8^x
→ (2²)^(2x−1) = (2³)^x
→ 2^(4x−2) = 2^(3x)
→ 4x − 2 = 3x
→ x = 2
Strategy 3 — Using Log Properties to Solve
Combine Logs → Convert → Solve
log₂(x) + log₂(x−2) = 3
→ log₂(x(x−2)) = 3   (product rule: combine)
→ 2³ = x(x−2)         (convert)
→ 8 = x² − 2x
→ x² − 2x − 8 = 0
→ (x−4)(x+2) = 0
→ x = 4 or x = −2

Check: x = −2 → log₂(−2) is undefined → reject x = −2
Answer: x = 4
❌ Always check for extraneous solutions. Logs require positive arguments. After solving, plug back in and verify the argument of every log in the original equation is positive. Reject any x that makes a log argument zero or negative.

Log Graph Features & Intercepts

Topic 2.13 · 🔥 Biggest Miss (5/10 wrong)
Key Features of f(x) = logb(x)

The parent log graph has predictable features that shift and transform with the same rules as any function:

FeatureParent: logb(x)With transformations: a·logb(x−h)+k
Domainx > 0x > h (shifted by horizontal shift h)
RangeAll reals (−∞, +∞)All reals — always, for any log
x-intercept(1, 0) — log of 1 is always 0Set f(x)=0 and solve. NOT automatically (1,0).
y-interceptNone (undefined at x=0)Set x=0 if 0 is in the domain. May exist if h<0.
Vertical Asymptotex = 0 (the y-axis)x = h (where the argument = 0)
BehaviorIncreasing if b>1, decreasing if 0<b<1Same, modified by vertical stretch/flip
Finding the x-intercept — The Q4 Miss

The most common error: assuming the x-intercept is always (1, 0). It's only (1, 0) for the parent function logb(x). Any transformation moves it.

How to Find the x-intercept of ANY log function
Set f(x) = 0 and solve for x. That's all.

f(x) = log₃(x − 4) + 2
0 = log₃(x−4) + 2
log₃(x−4) = −2
3^(−2) = x−4
1/9 = x−4
x = 4 + 1/9 = 37/9

The x-intercept is (37/9, 0) — NOT (1, 0).
The (1,0) rule only applies to the unshifted parent f(x)=logb(x).
Finding the y-intercept — When It Exists

Set x = 0, but first check: is x=0 in the domain? If the domain requires x > h and h ≥ 0, then x=0 is outside the domain → no y-intercept.

y-intercept Check
f(x) = log₂(x + 8): domain x > −8. x=0 is in domain.
f(0) = log₂(0+8) = log₂(8) = 3 → y-intercept at (0, 3)

g(x) = log₃(x − 2): domain x > 2. x=0 is NOT in domain.
→ No y-intercept. The graph doesn't cross the y-axis.

2B.5

Semi-log Plots

Topic 2.15 · 🔥 4/10 Wrong
What a Semi-log Plot Is

A semi-log plot has a regular (linear) x-axis but a logarithmic y-axis. The y-axis doesn't show y values directly — it shows log(y). This is used to linearize exponential data: if the original data is exponential, plotting log(y) vs x gives a straight line.

The Key Insight
If f(x) = a · bˣ (exponential), then:
log(f(x)) = log(a) + x·log(b)

This is Y = mx + c where Y = log(y), m = log(b), c = log(a)

Exponential data → straight line on a semi-log plot
The slope of the semi-log line = log(b) = log(growth factor)
Reading a Semi-log Plot — What Each Feature Means
On the Semi-log PlotWhat It Means for the Original Data
Straight lineOriginal data is exponential. A straight line on a semi-log plot is the signature of exponential growth/decay.
Slope of the lineslope = log(b) where b is the growth factor. To find b: b = 10^(slope)
y-intercept of the linelog(a) where a is the initial value. To find a: a = 10^(y-intercept)
Curved lineData is NOT purely exponential (could be polynomial or other)
⚠️ The AP question pattern: You're given a table of (x, log y) values or a semi-log graph. You're asked to find the original exponential model or to identify whether the data is exponential. If the (x, log y) pairs are linear (constant slope), the original data is exponential.
From Semi-log to Exponential Model
Full Process — Semi-log Table to Exponential Equation
Given: x = 0,1,2,3 and log(y) = 1.0, 1.5, 2.0, 2.5

Step 1: Check that log(y) values are linear in x. Differences: 0.5, 0.5, 0.5 ✓ → data is exponential.

Step 2: Slope = 0.5 = log(b) → b = 10^0.5 = √10 ≈ 3.162

Step 3: y-intercept = log(a) = 1.0 → a = 10¹ = 10

Step 4: Model: y = 10 · (√10)ˣ or equivalently y = 10^(1+0.5x)

Blind — Interpreting a Semi-log Statement
A semi-log plot of population (log P on y-axis, years on x-axis) shows a straight line with slope 0.04 and y-intercept 3.2.

(a) What does the slope tell you about population growth?
(b) What was the initial population?
(c) Write the population model P(t).
(d) In how many years will the population be 10 times the initial value?
(a) Slope = 0.04 = log(b) → b = 10^0.04 ≈ 1.096. The population grows by about 9.6% per year.

(b) y-intercept = 3.2 = log(a) → a = 10^3.2 ≈ 1585 people

(c) P(t) = 1585 · (10^0.04)ᵗ ≈ 1585 · (1.096)ᵗ

0.04t + 3.2 = 4.2 → 0.04t = 1 → t = 25 years
