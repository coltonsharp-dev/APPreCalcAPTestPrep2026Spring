Function Composition

Topic 2.7
What Composition Is — The Chain of Functions

Composition feeds the output of one function as the input into another. Written (f ∘ g)(x) = f(g(x)), it means: evaluate g first, then feed that result into f.

f(g(x)) — The Order of Operations
Input
x
→
Apply g first
g(x)
→
Feed into f
f(g(x))
→
Final Output
answer
Reading f(g(x)): the function closest to x is always applied first. g acts on x first, then f acts on the result.
❌ Order matters — composition is NOT commutative. f(g(x)) ≠ g(f(x)) in general. f(g(2)) means: find g(2) first, then apply f to that result. g(f(2)) means: find f(2) first, then apply g. Different order = different answer.
Three Ways Composition Appears on the AP
Type 1 — Analytical (from formulas)
f(x) = 2x + 1, g(x) = x² − 3
f(g(x)) = f(x²−3) = 2(x²−3) + 1 = 2x² − 5
g(f(x)) = g(2x+1) = (2x+1)² − 3 = 4x²+4x+1−3 = 4x² + 4x − 2
Type 2 — From a Table (Q6 / Q18 type — high miss rate)
x1234
f(x)5312
g(x)2413
f(g(2)) = f(4) = 2   ← g(2)=4 first, then f(4)=2
g(f(3)) = g(1) = 2   ← f(3)=1 first, then g(1)=2
f(f(2)) = f(3) = 1   ← f(2)=3 first, then f(3)=1
Type 3 — Real-World Composition (water ripples / nested context)
A stone dropped in water creates ripples. Radius: r(t) = 3t (feet). Area: A(r) = πr²
Area as function of time: A(r(t)) = A(3t) = π(3t)² = 9πt²

Read: "A composed with r" = "area in terms of time" = plug r(t) into A(r).

Using the table above, find: (a) f(g(1))   (b) g(f(4))   (c) f(g(g(2)))
1
(a) f(g(1)) — evaluate g(1) first, then f of that result

→ The innermost function acts first. g(1) gives us the input for f.
2
(b) g(f(4)) — evaluate f(4) first, then g of that result
f(4) = 2 (from table) → g(2) = 4 → g(f(4)) = 4
3
(c) f(g(g(2))) — work inside out: three layers
Layer 1 (innermost): g(2) = 4
Layer 2: g(4) = 3 (now apply g to the 4 we got)
Layer 3: f(3) = 1
f(g(g(2))) = 1
→ Three-layer composition: always evaluate from the inside out. Each output becomes the next input.

2C.2

Inverse Functions — All Three Representations

Topic 2.8 · 🔥 Highest Miss Rate in Unit 2
What an Inverse Does

The inverse function f⁻¹ reverses what f does: if f takes input a to output b, then f⁻¹ takes input b back to output a. The input and output swap roles.

The Core Property
f(a) = b  ↔  f⁻¹(b) = a

f(f⁻¹(x)) = x  and  f⁻¹(f(x)) = x  ← they cancel each other

Domain of f = Range of f⁻¹. Range of f = Domain of f⁻¹.
Inverse from a Table — The Most Missed

To find f⁻¹ from a table: swap the x and y columns. Every row (x, y) in the f table becomes (y, x) in the f⁻¹ table.

Table Swap Method
Table for fx1234
f(x)7593
Table for f⁻¹x7593
f⁻¹(x)1234
f⁻¹(7) = 1   f⁻¹(5) = 2   f⁻¹(9) = 3   f⁻¹(3) = 4
The output column of f becomes the input column of f⁻¹.
Graphical Inverse — Reading f⁻¹ from a Graph of f

The graph of f⁻¹ is the reflection of the graph of f over the line y = x. Every point (a, b) on f's graph becomes (b, a) on f⁻¹'s graph.

f (blue) and f⁻¹ (green dashed) are reflections across y=x. Point (a,b) on f becomes (b,a) on f⁻¹. To read f⁻¹(b) from a graph of f: find b on the y-axis of f, read the corresponding x — that's your answer.
How to Read f⁻¹(b) from a Graph of f — Step by Step
To find f⁻¹(5) using a graph of f:
Step 1: Find where f(x) = 5 on the graph (find y=5 on the curve)
Step 2: Read the x-value at that point
Step 3: That x-value is f⁻¹(5)

Example: if the graph shows f(3) = 5, then f⁻¹(5) = 3.
You are NOT reading the y-value from the inverse graph — you are finding where f equals the input you're given.
Inverse Algebraically — Swap and Solve
    1
    Write y = f(x)
    2
    Swap x and y — this is the key step that reverses input/output
    3
    Solve for y — isolate y on one side
    4
    Rename y as f⁻¹(x). State the domain restriction if needed.
Verifying an Inverse — Composition Check

Two functions are inverses if and only if f(g(x)) = x AND g(f(x)) = x for all x in the domain. You must check both compositions.

Verification Example
f(x) = 3x − 5,   g(x) = (x+5)/3

f(g(x)) = 3·x+53 − 5 = (x+5) − 5 = x ✓
g(f(x)) = (3x−5)+53 = 3x3 = x ✓

Both equal x → confirmed inverses.

(a) Find f⁻¹(3), f⁻¹(7), and f⁻¹(9) from the graph.
(b) Find the x-intercept of f⁻¹.

Blind
(a) Find f⁻¹(x) for f(x) = (2x−1)/(x+3) using the swap-and-solve method.
(b) Verify using composition: compute f(f⁻¹(x)) and confirm it equals x.
(a) y = (2x−1)/(x+3). Swap: x = (2y−1)/(y+3)
x(y+3) = 2y−1
xy+3x = 2y−1
xy−2y = −1−3x
y(x−2) = −(1+3x)
y = −(1+3x)/(x−2) = (3x+1)/(2−x)

(b) f(f⁻¹(x)) = f((3x+1)/(2−x)) = [2·(3x+1)/(2−x)−1] / [(3x+1)/(2−x)+3]

Denominator: (3x+1)/(2−x) + 3 = (3x+1+6−3x)/(2−x) = 7/(2−x)
f(f⁻¹(x)) = [(7x)/(2−x)] / [7/(2−x)] = 7x/7 = x ✓

Residuals & Model Validation

Topic 2.6 · 🔥 5/10 Wrong on Practice Exam
What a Residual Is

A residual is the difference between the actual observed value and the value predicted by the model. It measures how well the model fits at a specific point.

Residual Formula
Residual = Actual value − Predicted value
= yobserved − ymodel

Positive residual: actual > predicted → model underestimates
Negative residual: actual < predicted → model overestimates
❌ Order matters in the formula. Residual = actual MINUS predicted. Not predicted minus actual. A positive residual means the model fell short (data is above the model line). A negative residual means the model went too high.
Residual Plots — Reading Them to Validate a Model

A residual plot shows the residuals on the y-axis vs the x-values (or predicted values) on the x-axis. The pattern in the residual plot tells you whether the model is appropriate.

Good Fit — Random Scatter
Model is appropriate ✓
Points scattered randomly above and below zero. No pattern. Model type fits the data well.
Bad Fit — Curved Pattern
Model is NOT appropriate ✗
U-shape or arch pattern visible. Systematic error — try a different model type (exponential instead of linear, etc.)
💡 The AP question pattern for residuals: You're given a residual plot and asked "Does the linear/exponential model appear appropriate?" Answer based on the shape. Random scatter → yes. Any curve, U-shape, fan, or trend → no, and name what you see.
