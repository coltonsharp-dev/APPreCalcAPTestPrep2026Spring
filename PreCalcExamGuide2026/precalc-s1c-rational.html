Rational Function Anatomy

Topics 1.7–1.11
The Structure

A rational function has the form f(x) = p(x)/q(x) where p and q are polynomials. The denominator controls the interesting behavior — where it equals zero is where asymptotes or holes appear.

Always Factor First
f(x) = x² − x − 6x² − 4 = (x−3)(x+2)(x+2)(x−2)

After factoring: shared factors → holes. Remaining denominator factors → vertical asymptotes.
Holes vs Vertical Asymptotes — The Core Distinction

Both holes and vertical asymptotes occur where the denominator equals zero. The difference is whether the numerator also equals zero at that x-value (i.e., whether it's a shared factor).

Hole
(x − c) cancels
Factor appears in BOTH numerator and denominator. Cancel it → the hole's x-coordinate. Substitute into simplified form → hole's y-coordinate. Graph is undefined there — a literal open circle.
Vertical Asymptote
x = c (doesn't cancel)
Factor appears in denominator ONLY after canceling. Set that factor = 0. The graph approaches ±∞ at that x-value. A vertical line the graph never crosses.
Horizontal Asymptote
y = limit as x→±∞
Determined by degree comparison between numerator and denominator. Three cases based on degrees.
Vertical asymptote (red dashed) at x=−2: denominator factor that doesn't cancel. Hole (orange circle) at x=2: factor that cancels from both. Horizontal asymptote (blue dashed) at y=1: degree comparison.
Horizontal Asymptote — Three Cases
HA Rules — Compare Degrees of Numerator (n) and Denominator (d)
n < d  →  HA at y = 0           ← denominator "wins"
n = d  →  HA at y = leading coeff ratio  ← degrees tie → coefficients decide
n > d  →  No HA (slant or none)        ← numerator "wins"

Example: f(x) = (3x²+1)/(x²−4) → n=d=2 → HA at y = 3/1 = 3
Slant (Oblique) Asymptote

When the numerator degree is exactly 1 more than the denominator degree, there is a slant asymptote. Find it by performing polynomial long division — the quotient (ignoring the remainder) is the slant asymptote equation.

Slant Asymptote — Long Division
f(x) = x² + 3x − 2x − 1

Numerator degree (2) = Denominator degree (1) + 1 → slant asymptote exists

Divide: x² + 3x − 2 ÷ (x−1) = x + 4 remainder 2

Slant asymptote: y = x + 4 (ignore the remainder 2/(x−1) → goes to 0 as x→∞)
💡 Quick check: If degree(numerator) = degree(denominator) + 1 → slant asymptote. If degree(numerator) = degree(denominator) + 2 or more → no asymptote at all (graph curves away). Only the "exactly 1 more" case gives a slant.
Step-by-Step: Analyzing Any Rational Function
1
Factor numerator and denominator completely.
2
Holes: Identify factors that cancel. Set each = 0 for x-coord. Plug into simplified form for y-coord.
3
Vertical asymptotes: Remaining denominator factors after canceling. Set each = 0.
4
Horizontal/slant asymptote: Compare degrees (n vs d). Apply the three HA cases or do long division.
5
Zeros: Numerator factors only (after canceling) set = 0.

Function Transformations

Topic 1.12 · Watch
The Complete Transformation Reference

Starting from a parent function f(x), every transformation modifies the graph in a predictable way. The key is knowing which operations affect the input (horizontal) vs the output (vertical).

TransformationNotationEffect on GraphDirection
Vertical shift up kf(x) + kMove graph UP k unitsVertical (output)
Vertical shift down kf(x) − kMove graph DOWN k unitsVertical (output)
Horizontal shift right hf(x − h)Move graph RIGHT h unitsHorizontal (input)
Horizontal shift left hf(x + h)Move graph LEFT h unitsHorizontal (input)
Vertical stretch by aa·f(x), a > 1Stretch away from x-axisVertical (output)
Vertical compress by aa·f(x), 0 < a < 1Compress toward x-axisVertical (output)
Horizontal compress by bf(bx), b > 1Compress toward y-axisHorizontal (input — opposite!)
Horizontal stretch by bf(bx), 0 < b < 1Stretch away from y-axisHorizontal (input — opposite!)
Reflect over x-axis−f(x)Flip verticallyVertical (output)
Reflect over y-axisf(−x)Flip horizontallyHorizontal (input)
❌ The horizontal direction trap: f(x + 3) shifts LEFT (not right). f(x − 3) shifts RIGHT (not left). f(2x) compresses (not stretches). Input transformations always do the opposite of what they look like. This is the most common transformation error.
Transformations from a Table — The Q26 Type

The AP exam frequently gives you a table of values for f(x) and asks you to find values for a transformed function. The key: figure out which input to use for f based on the transformation.

Strategy — Working Backwards from the Input
If you need g(x) = f(x − 2) at x = 5: you need f(5 − 2) = f(3) — look up x=3 in the table

If you need g(x) = 2f(x) + 1 at x = 4: you need 2·f(4) + 1 — look up x=4, multiply by 2, add 1

If you need g(x) = f(2x) at x = 3: you need f(2·3) = f(6) — look up x=6 in the table

The Binomial Theorem

Topic 1.11
The Formula

The Binomial Theorem expands (a + b)ⁿ as a sum of terms. Each term uses a binomial coefficient — the number of ways to choose k items from n.

Binomial Theorem
(a + b)ⁿ = C(n,0)aⁿ + C(n,1)aⁿ⁻¹b + C(n,2)aⁿ⁻²b² + ··· + C(n,n)bⁿ

where C(n,k) = n!k!(n−k)!  ← "n choose k"

The kth term (starting at k=0) is: C(n,k) · aⁿ⁻ᵏ · bᵏ
Pascal's Triangle — The Fast Way

The binomial coefficients for each row of (a+b)ⁿ come directly from Pascal's Triangle. Each number is the sum of the two numbers above it.

Pascal's Triangle (rows 0–5)
n=0:                1
n=1:             1   1
n=2:          1   2   1
n=3:       1   3   3   1
n=4:    1   4   6   4   1
n=5: 1   5   10   10   5   1
💡 AP exam shortcut: For small n (2,3,4,5), use Pascal's Triangle directly. For larger n or when asked for a specific term only, use C(n,k) directly. You almost never need to expand the whole thing — the AP usually asks for a specific term or coefficient.
Finding a Specific Term — The AP Question Type
Specific Term Example
Find the x³ term in the expansion of (2x + 3)⁵

The x³ term has b = 3x has power k → we want x³ so k = 3 (b = 3, a = 2, n = 5... wait)
Actually: a = 2x, b = 3, n = 5. We want the term where (2x) gives us x³ → (2x)³ → k = 3 (counting from the b side, so the a side gets 5−3=2)

Wait, let's be precise: general term = C(5,k)·(2x)^(5−k)·3^k
For x³: 5−k = 3 → k = 2
Term = C(5,2)·(2x)³·3² = 10 · 8x³ · 9 = 720x³
